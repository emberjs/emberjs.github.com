DS.PromiseObject Class packages/ember-data/lib/system/store.js:1504

Extends: Ember.ObjectProxy

Uses: Ember.PromiseProxyMixin

Defined in: packages/ember-data/lib/system/store.js:1504

Module: ember-data

A PromiseObject is an object that acts like both an Ember.Object and a promise. When the promise is resolved the the resulting value will be set to the PromiseObject's content property. This makes it easy to create data bindings with the PromiseObject that will be updated when the promise resolves.

For more information see the Ember.PromiseProxyMixin documentation.

var promiseObject = DS.PromiseObject.create({
  promise: $.getJSON('/some/remote/data.json')
});

promiseObject.get('name'); // null

promiseObject.then(function() {
  promiseObject.get('name'); // 'Tomster'
});
